Instructions and Details:



Quick summary:

The game is simple, enter the correct username and password to win points. Once a correct username and password have been entered, a new username and password are created. The game ends when the administrator ends the game or when all the correct usernames and passwords have been entered. The player with the most points wins.

Near the top of the page the current loged in user's name and points for the current game are displayed. Note: if no game is in progress the points will show 0.

While a game is in progress there will be a field for a username, a field for a password, and a login button displayed. During a game the players enter usernames and passwords into the fields click the login button (or press enter if they are focused on one of the fields) to see if they are correct. If the username and password are correct the player will be awarded points and the game will continue. If the username and password are incorrect the player will not be awarded points and the game will continue.

At the end of the game, the player with the most points wins.

The game can end in one of two ways:

  1. The game ends when the administrator ends the game.
  2. The game ends when all the correct usernames and passwords have been entered.


Hints


Hints are super important to being successful at this game.

The hints are displayed in the "Current password details" section of the page. The hints are updated every time a player enters a correct username and password and a new password is created. The hints are updated in real time, so if you are playing the game and you see the hints change, you know that someone else has entered a username and password and the hints have changed.


The hints let the player know details about the currrent username and password. They can show the length of the username or password, the characters used in the username or password, or other details about the username or password. Watching this section is VITAL to winning the game.
